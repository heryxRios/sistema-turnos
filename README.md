# sistema-turnos
SISTEMA DE TURNOS PARA USUSARIOS EN SALA DE ESPERA; CONTIENE INTERFAZ DE ADMINISTRADOR GLOBAL, INTERFAZ DE USUARIOS PARA ATENCIÓN EN VARIAS VENTANILLAS O TOTEMS QUE GENERAN LOS TURNOS QUE POSTERIORMENTE LOS USUARIOS LOS VERAN EN UNA INTERFAZ EN PATANLLA EN SALA DE ESPERA, DONDE SE LES INDICARA EN QUE VENTANILLA ACUDIR Y SU SU TURNO ES GENERAL O PRIORITARIO (PERSONAS CON DISCAPACIDAD O TERCERA EDAD).
